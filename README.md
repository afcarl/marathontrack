marathontrack
=====

Dependencies:
* Flask (pip install Flask)
* MySQL python (pip install MySQL-python)

To run the website (after installing dependencies), run:

	$ python site.py
	
Then open your browser to [`http://localhost:5000/`](http://localhost:5000/) to see the result.
